## Description
오늘하나는 사용자가 직접 입력한 할 일과 피드백을 바탕으로 지금 할 일 하나를 추천하는 로컬 우선 생산성 앱입니다.

- 앱을 열면 가장 먼저 Now Card로 지금 할 일 하나를 보여줍니다.
- 중요도, 마감, 예상 소요시간, 선호 시간대, 미룸 기록을 함께 반영합니다.
- 완료, 10분 뒤, 오늘 말고, 별로예요 피드백으로 추천이 더 정교해집니다.
- 카톡, 메일, 메모 문장을 직접 붙여넣으면 할 일 후보로 바꿔드립니다.
- 홈 화면 위젯과 로컬 알림으로 다시 돌아오기 쉬운 흐름을 만듭니다.
- 모든 데이터는 현재 기기 안에만 저장되며, 로그인과 외부 서버가 없습니다.

오늘하나는 화려한 AI 채팅 앱이 아니라, “지금 뭐 해야 하지?”라는 망설임을 줄이고 바로 실행으로 이어지게 돕는 하루 비서입니다.
