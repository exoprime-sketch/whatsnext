# 개인정보 처리방침

## 문서 정보
- 서비스명: 오늘하나 (WhatsNext)
- 운영 주체: `YOUR_COMPANY_NAME`
- 문의 이메일: `support@example.com`
- 시행일: `YYYY-MM-DD`

## 1. 수집하는 정보
오늘하나는 사용자가 직접 입력하거나 직접 붙여넣은 정보만 기기 안에 저장합니다.

- 할 일 제목
- 메모
- 중요도, 마감, 선호 시간대
- 완료/미룸/별로예요 같은 앱 내 피드백
- 알림 시간 설정
- 로컬 사용 기록 지표

## 2. 저장 위치
모든 데이터는 사용자의 기기 안에만 저장됩니다. 기본 버전에서는 외부 서버나 제3자 서비스로 전송하지 않습니다.

## 3. 외부 서버 전송 여부
오늘하나 v1.0은 로그인, 계정 동기화, 외부 백엔드 저장 기능을 사용하지 않습니다. 따라서 사용자가 입력한 데이터는 기기 밖으로 전송되지 않습니다.

## 4. 자동으로 수집하지 않는 정보
오늘하나는 다음 정보를 자동 수집하거나 접근하지 않습니다.

- 위치 정보
- 연락처
- 사진 및 미디어 라이브러리
- 건강 정보
- 광고 식별자
- 브라우징 이력
- 문자/메신저/메일 내용 자동 읽기
- 키보드 입력 감시

## 5. Capture 기능 안내
Capture 기능은 사용자가 직접 복사해서 붙여넣은 텍스트만 처리합니다. 문자, 메일, 메신저 앱의 내용을 자동으로 읽거나 가져오지 않습니다.

## 6. 알림 사용 방식
사용자가 직접 시간대를 선택한 경우에만 로컬 알림을 사용합니다. 알림 내용은 기기 안에 저장된 할 일 데이터만 바탕으로 생성됩니다.

## 7. 로컬 사용 기록
앱은 외부 전송 없이 다음과 같은 로컬 사용 기록을 기기 안에만 저장할 수 있습니다.

- 첫 실행 시각
- 온보딩 완료 시각
- 첫 할 일 생성 시각
- 완료/미룸/건너뜀 횟수
- Capture 사용 횟수
- 마지막 활성 시각
- 활성 일수

이 정보는 앱 내부 UX 개선과 사용 기록 요약 표시에만 사용됩니다.

## 8. 데이터 삭제 방법
앱 설정의 `전체 데이터 삭제`를 누르면 할 일, 피드백 로그, 알림 설정, 로컬 사용 기록을 포함한 앱 데이터가 현재 기기에서 삭제됩니다.

## 9. 향후 기능 확장에 대한 안내
향후 AI API, 외부 분석 SDK, 계정 동기화, 백업 기능이 추가되는 경우, 수집/이용 목적과 전송 범위를 별도로 고지하고 필요한 동의를 받겠습니다.

## 10. 문의
개인정보 처리에 관한 문의는 `support@example.com`으로 받을 예정이며, 출시 전 실제 운영 정보로 교체해야 합니다.
