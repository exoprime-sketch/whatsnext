# Review Notes

## Product Summary
오늘하나는 사용자가 직접 입력한 할 일과 직접 붙여넣은 텍스트만 바탕으로, 지금 할 일 하나를 추천하는 로컬 우선 생산성 앱입니다.

## Important Review Notes
- 로그인 기능이 없습니다.
- 결제 기능이 없습니다.
- 외부 서버 전송이 없습니다.
- 광고 SDK가 없습니다.
- 사용자 추적이 없습니다.
- Capture 기능은 사용자가 직접 붙여넣은 텍스트만 처리합니다.
- 문자, 메일, 메신저 내용을 자동 수집하거나 읽지 않습니다.
- 알림은 UserNotifications 기반의 로컬 알림만 사용합니다.
- 위젯은 App Group 저장소의 로컬 snapshot만 표시합니다.

## Reviewer Test Steps
1. 앱을 실행합니다.
2. 온보딩에서 알림 시간대를 고르거나 `나중에`를 선택합니다.
3. 첫 할 일을 입력하고 `첫 추천 만들기`를 누릅니다.
4. Today 화면에서 Now Card가 보이는지 확인합니다.
5. `완료`, `10분 뒤`, `오늘 말고`, `별로예요` 버튼 중 하나를 눌러 반응을 확인합니다.
6. `캡처` 탭에서 텍스트를 직접 붙여넣고 후보를 저장해봅니다.
7. `설정` 탭에서 개인정보 안내, 데이터 삭제, 알림 시간 변경을 확인합니다.
